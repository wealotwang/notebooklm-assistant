# Gemini 界面修复与功能增强计划 (修订版)

根据您的建议，我们调整了侧边栏的处理策略。

## 1. 移除红色调试框 (Red Box)
*   **解决方案**: 在 `content_gemini.js` 中彻底删除 `showDebugBox` 函数及其调用，清除界面上的红色调试元素。

## 2. 优化侧边栏折叠表现 (Sidebar Squeeze)
*   **策略调整**: **在侧边栏收起时不显示文件夹**。
*   **解决方案**:
    *   **状态监听**: 在 `content_gemini.js` 中使用 `ResizeObserver` 监听侧边栏宽度。
    *   **自动隐藏**: 当侧边栏宽度小于特定阈值（如 150px）时，直接隐藏整个文件夹区域 (`display: none`)。这能避免布局挤压，保持界面清爽，待侧边栏展开后自动恢复显示。

## 3. 修复时间轴滚动条显示异常 (Timeline Scrollbar)
*   **解决方案**:
    *   **位置微调**: 调整 CSS `right` 值，防止与原生滚动条冲突。
    *   **高度适配**: 修正时间轴容器的高度计算，确保滑块（Slider）能正确映射当前阅读进度。

## 4. 实现文件夹重命名功能 (Folder Renaming)
*   **解决方案**:
    *   **功能移植**: 将 NotebookLM 的双击重命名功能移植到 Gemini。
    *   **交互细节**: 支持双击文件夹名称进入编辑模式，回车或点击外部保存修改。
