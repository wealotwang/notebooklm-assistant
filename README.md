# Google AI Companion

![Version](https://img.shields.io/badge/version-3.0.0.15-blue)

A powerful Chrome extension that supercharges your Google AI experience.

## Features

### NotebookLM
*   **Folder Management**: Organize your sources into custom folders.
*   **Folder Renaming**: Double-click folder names to rename them.
*   **Timeline Navigation**: Visual timeline for quick navigation of long chat histories.
*   **Batch Operations**: Move or delete multiple sources at once.
*   **Source Isolation**: Keep folders separate for each notebook.
*   **Real-time Sync**: Automatic UI updates when sources are modified.

### Gemini
*   **Timeline Navigation**: Visual timeline for quick navigation of long chat histories.

## Development
*   **Packaging**: Use `sh package_extension.sh` to create a clean zip for the Chrome Web Store.
*   **Documentation**: Process docs and archives are located in `dev_docs/`.

## Installation
1. Clone this repository.
2. Open `chrome://extensions/` in Chrome.
3. Enable "Developer mode".
4. Click "Load unpacked" and select this directory.
